# WeatherCast

## :gear: Features
- [ ] Search the weather by the city's name;
- [ ] Show the city's weather;
- [ ] Show nex day's weather;
- [ ] Show humidity, wind pressure, cloud porcentage, etc. 

## :computer: Instalation
```bash
$ git clone http://github.com/SitaGomes/WeatherCast

$ cd WeatherCast

$ npm install

$ npm start

```
