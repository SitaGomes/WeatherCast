## :computer: Instalation
```bash
// First clone the repository
$ git clone http://github.com/SitaGomes/WeatherCast

// Them go to the project folder
$ cd WeatherCast

// Install all the dependencies
$ npm install

// Run the project and 
$ npm start

```
